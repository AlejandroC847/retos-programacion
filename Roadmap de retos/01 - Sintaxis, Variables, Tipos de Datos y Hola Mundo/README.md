# Ejercicio 01 - Sintaxis, Variables, Tipos de Datos y Hola Mundo

## Descripción

- Crea un comentario en el código y coloca la URL del sitio web oficial del
  lenguaje de programación que has seleccionado.
- Representa las diferentes sintaxis que existen de crear comentarios
  en el lenguaje (en una línea, varias...).
- Crea una variable (y una constante si el lenguaje lo soporta).
- Crea variables representando todos los tipos de datos primitivos
  del lenguaje (cadenas de texto, enteros, booleanos...).
- Imprime por terminal el texto: "¡Hola, [y el nombre de tu lenguaje]!"

## Enfoque

Este módulo intruduce conceptos básicos de creación de comentarios simples y multilínea;
asi como de creación de constantes y variables de múltiples tipos primitivos de dato.
Finalmente imprime en pantalla el mensaje de bienvenida a la programación con Python:
"_Hola, Python!_"

## Estructura

- sin_var_tip_hola_mundo.py → lógica principal
- tests.py → pruebas básicas con assert y pytest

## Ejecución

### Ejecutar Programa

```bash
python sin_var_tip_hola_mundo.py
```

### Ejecutar Tests

Este módulo no requiere de test.
