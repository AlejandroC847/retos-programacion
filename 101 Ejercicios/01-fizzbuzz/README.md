# Ejercicio 01 - FizzBuzz

## Descripción

Escribe un programa que muestre por consola los números del 1 al 100, sustituyendo:

- Múltiplos de 3 por "fizz"
- Múltiplos de 5 por "buzz"
- Múltiplos de 3 y 5 por "fizzbuzz"

---

## Enfoque

Se implementa una función `fizzbuzz(n)` que evalúa un número y devuelve el resultado correspondiente como string.

Luego, se utiliza una función `ejecutar_fizzbuzz()` para imprimir la secuencia completa.

---

## Estructura

- `fizzbuzz.py` → lógica principal
- `tests.py` → pruebas básicas con `assert`

---

## Ejecución

### Ejecutar programa

``` bash
python fizzbuzz.py
```

### Ejecutar tests

```bash
python tests.py
