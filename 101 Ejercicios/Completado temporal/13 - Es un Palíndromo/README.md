# Ejercicio 13 - Es un Palíndromo?

## Descripción

Escribe una función que reciba un texto y retorne verdadero o falso (Boolean)
según sean o no palíndromos. Un Palíndromo es una palabra o expresión que es
igual si se lee de izquierda a derecha que de derecha a izquierda.
NO se tienen en cuenta los espacios, signos de puntuación y tildes.

## Enfoque

Se implementa una función `is_palindrome()` que recibe una cadena de texto y
evalua si se trata de un palindromo o no, retornando un valor booleano de esa evaluación.

## Estructura

- es_un_palindromo.py → lógica principal
- tests.py → pruebas básicas con assert y pytest

## Ejecución

### Ejecutar Programa

```bash
python es_un_palindromo.py
```

### Ejecutar Tests

```bash
python tests.py
```

ó

```bash
pytest test.py
```
